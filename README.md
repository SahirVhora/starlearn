# StarLearn ⭐

A free, fun learning platform for UK Year 4-6 primary school students.

**Live:** https://sahirvhora.github.io/starlearn

## Subjects
- 🔢 Maths — Place Value, Multiplication & Division, Fractions, Geometry
- 📚 English — Grammar, Punctuation, Spellings, Comprehension
- 🔬 Science — Living Things, States of Matter, Electricity, Food Chains

## Features
- KS2 curriculum aligned (Year 4-6)
- 5 questions per topic, shuffled each time
- AI tutor hints powered by Grok (xAI) — explains concepts in simple language
- Score tracking saved in browser
- Works on phone and desktop

## Setup
1. Open `index.html` in any browser — no install needed
2. Click ⚙️ Settings and paste your Grok API key (get one free at https://console.x.ai)
3. Pick a subject and start learning!

## Tech
Pure HTML/CSS/JS — no frameworks, no build step. Single page app.
AI hints use Grok API (`grok-3-mini`) via `https://api.x.ai/v1/chat/completions`.
